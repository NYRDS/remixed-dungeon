====== Assassin ======

===== Description =====
The Assassin is a mastery path available for certain hero classes in Remixed Dungeon. This subclass offers specialized abilities and mechanics.

===== Abilities =====
The Assassin subclass provides unique abilities and bonuses:


The Assassin gains bonus damage for attacks from stealth or invisibility.

===== Unlocking =====
The Assassin subclass is unlocked by achieving Mastery with the appropriate class by using the Tome of Mastery. This is typically done by reaching level 10 with the base class.
