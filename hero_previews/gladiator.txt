====== Gladiator ======

===== Description =====
The Gladiator is a mastery path available for certain hero classes in Remixed Dungeon. This subclass offers specialized abilities and mechanics.

===== Unlocking =====
The Gladiator subclass is unlocked by achieving Mastery with the appropriate class by using the Tome of Mastery. This is typically done by reaching level 10 with the base class.
