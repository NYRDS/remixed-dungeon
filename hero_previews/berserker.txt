====== Berserker ======

===== Description =====
The Berserker is a mastery path available for certain hero classes in Remixed Dungeon. This subclass offers specialized abilities and mechanics.

===== Abilities =====
The Berserker subclass provides unique abilities and bonuses:


The Berserker enters a fury state when below 50% health, gaining increased damage.

===== Unlocking =====
The Berserker subclass is unlocked by achieving Mastery with the appropriate class by using the Tome of Mastery. This is typically done by reaching level 10 with the base class.
